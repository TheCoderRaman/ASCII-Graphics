Welcome to my project. It is a simple console based 2d graphics system that includes loading images, placing pixels, an image editor, and includes some basic shapes and GUI systems. It also handles user input using windows messages. It is still in the works to make things more pretty, because lets face it, my code is a bit ugly. You include a single header and get a ton of different usefull tools.
